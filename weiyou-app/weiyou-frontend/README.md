# 微友前端联调说明

## 本地启动

```bash
npm install --legacy-peer-deps
npm run dev:h5
```

## 生产构建

```bash
npm run build:h5
```

## Docker Compose

在 `weiyou-app` 目录执行：

```bash
docker compose up --build
```

- H5 访问地址：`http://localhost:8088`
- Docker 部署下前端会通过 Nginx 自动把 `/api` 与 `/ws` 代理到 gateway

## WebSocket

- 前端会在登录成功后自动连接统一网关地址，开发态默认 `ws://localhost:8081/ws/chat`
- 若需要自定义地址，可通过 `VITE_WS_BASE_URL` 覆盖默认 WebSocket 地址
- 连接参数会自动带上 `accessToken` 与 `deviceId`
- 聊天页顶部会显示实时连接状态
- 连接异常时会自动按退避策略重连，恢复后会重新刷新会话和当前聊天页消息
- 消息发送失败时会保留失败态，支持点击重发；HTTP 与 WebSocket 都不可用时会进入待重试队列
- 待重试消息会持久化到本地，App 重启后仍可恢复，并在重连成功后继续尝试发送
- 会话页支持时间分组显示、自动滚动到底部，以及按会话维度自动保存输入草稿
- 会话页已支持图片消息与文件消息，发送前会先走后端 `/api/media/upload/local` 本地上传链路，再发送可访问 URL
- 会话页已支持语音消息录制与播放；当前 H5 端通常不支持 `getRecorderManager`，更适合在 App / 小程序端联调
- 会话页已支持视频消息发送与播放；当前视频同样会先上传，再发送视频 URL 与时长元数据
- 会话输入区已支持拍照发图、位置消息、联系人名片消息和表情快捷发送
- 会话页已支持多选消息、撤回自己发送的消息、转发到其他会话，以及快捷收藏到收藏页
- 会话页已支持引用回复、会话内消息搜索，以及单条消息本地置顶
- 会话设置页已支持置顶会话、免打扰、标为未读与清空聊天记录
- 联系人资料页支持动态打开单聊会话；新的朋友通过后会自动进入通讯录列表
- 群聊模块已支持群列表、建群、群详情、群成员列表、群公告编辑、全员禁言、入群方式设置、管理员设置、单成员禁言、群昵称设置、移除成员、转让群主和退出群聊
- 卡包页支持按类型筛选并直接核销卡券；表情商店支持下载、卸载和切换当前使用表情包
- 扫一扫页支持手动解析、摄像头扫码（平台支持时）与本地扫码记录；付款码可直接跳转转账页并带入目标用户
- 钱包模块已支持独立账单明细页，可按账单类型和日期范围筛选交易记录
- 设置中心已支持消息通知与隐私设置页，通知、声音、震动、加好友方式和位置可见性可直接保存到后端

## 启动配置

- App 启动时会先请求 `/app/bootstrap`
- 若后端返回公告、协议版本或版本升级信息，前端会按内容弹出一次启动提示
- 业务接口返回 `401` 时，会先尝试自动刷新 token，再重试一次原请求

- 当前 H5 构建已验证通过
